HTML - Hyper Text Markup Language

There are various types of tag. Here we describe 5 types of Tag.

  1. Simple Tag. Have opening and closing tag. Example- '<'p'>'<' /'p'>', '<'b'>'<' /'b'>',' <'h1'>'<' /'h1'>'' .. '<'h6'>'<' /'h6'>'
  2. Tag with attribute . Example- '<'form action="">'<' /form>'
  3. Empty Tag . Tag with attribute but have no closing tag. Example- '<'input type="text">', ' <'br'>', '<'img src="" alt="">'
  4. Container Tag . Have some another tags into container tag. Example- '<'head>'<' /head>',' '<'ol>'<' /ol>',' '<'ul>'<' /ul>'
  5. DOCTYPE Tag . Special Tag for HTML5. Example- '<'!DOCTYPE html>'

Some essential Tag list are stated below:

  • '<'link rel="icon" href="favicon.ico" type="image/x-icon">' - favicon link Tag.
  • '<'a href="">'<' /a>' - html links are used with the anchor tag.
  • '<'strong>'<' /strong>','<'p>'<' /p>','<'b>'<' /b>','<'i>'<' /i>','<'small> For Small text'<' /small>'
  • '<'div> Container tag. Hold one or more another tags. '<' /div>'
  • '<'ol>Ordered List. Example- 1,2,3,4,5'<' /ol>', '<'li> Item 1'<' /li>'
  • '<'ul>Unordered List. Example- ******'<' /ul>', '<'li> Item *'<' /li>'
  • '<'button value='Submit'>'<' /button>'
  • '<'img src="" alt="">'- Have no closing tag.
